# Martius Font Installation

To use the Martius font in the Spendo logo:

1. Download the Martius font from Fontspace:
   https://www.fontspace.com/martius-font-f137510

2. Extract the font files from the ZIP archive.

3. Copy the following font files to this directory (`public/fonts/`):
   - `Martius-Regular.ttf` (or similar regular weight file)
   - `Martius-Bold.ttf` (or similar bold weight file)

4. If the font files have different names, update the `@font-face` declarations in `src/index.css` to match the actual filenames.

5. The font will automatically load when you run the application.

Note: The font files are not included in the repository due to licensing. You must download them separately from Fontspace.
